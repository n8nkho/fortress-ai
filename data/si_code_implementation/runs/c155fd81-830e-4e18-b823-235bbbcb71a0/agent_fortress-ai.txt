## Behavior

1. **Entry block counting** — On each swarm cycle, when `side=flat` and reasoning matches:
   - `manual_denylist` → `denylist`
   - `pause_entries` / `swarm_session_critical_pause` / `pause_*` → `pause_entries`
   - `pattern_disabled:*` → `pattern_disables`

2. **Session summary** — `build_portfolio_session_metrics()` calls `generate_summary()` and attaches `entry_block_breakdown` to portfolio metrics.

3. **Diagnostic logging** — When `session_exit_count == 0` and `benchmark_change_1d_pct > 0`, logs at info:
   ```
   Entry blocks active: denylist=3, pause_entries=0, pattern_disables=5
   ```
